Real Examples
Ask anything about your phone
One command. Real data from your device.
📷 Camera
novax ask "Take a selfie and describe it"
Takes a front-camera photo → sends to AI → returns a detailed visual description
📍 GPS
novax ask "Where am I right now?"
Gets GPS coordinates → reverse geocodes → returns city, area, Google Maps link
💬 SMS
novax ask "Summarize my last 5 messages"
Reads recent SMS → AI summarizes key info, senders, and action items
🔋 Battery
novax ask "Is my battery healthy?"
Returns level, charging status, health rating, and temperature with AI analysis
📋 Clipboard
novax ask "Fix grammar in my clipboard"
Reads clipboard → fixes grammar → pastes improved text back
🔦 Torch
novax ask "Flash the torch 3 times"
Toggles the flashlight on and off 3 times with a 500ms interval
Get Started
Install in 3 steps
Works on any Android phone with Termux — no root needed.
STEP 1
Install Termux
From F-Droid only (not Play Store)
↗ F-Droid
STEP 2
Run installer in Termux
$ curl -fsSL https://raw.githubusercontent.com/lekhanpro/nova-agent/main/install.sh | bash
STEP 3
Configure & start
$ novax configure
$ novax ask "Hello!"
