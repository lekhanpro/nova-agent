Why Nova Agent
Most "AI on Android" is just
a chatbot in a terminal
Nova Agent gives the AI real access to your hardware — via Termux:API. Your data never leaves your device.
📱
Native Android hardware
Camera, GPS, SMS, sensors, volume, brightness, call log — real Termux:API commands, not web APIs. The AI actually controls your phone.
🔒
Completely private
Everything runs on your device. Photos stay local. SMS is read locally. No cloud processing. Only your prompts go to the AI API.
🧠
Persistent memory
Conversation history saved to ~/.nova_agent/history.json. The agent remembers context across sessions. 50-message rolling window.
🆓
Free to start
Google Gemini 2.0 Flash is completely free. Get your API key in 60 seconds at aistudio.google.com — no credit card required.
⚡
Streaming responses
OpenAI responses stream token-by-token as they generate. No waiting for the full answer — you see output in real time.
🔧
3 AI providers
OpenAI (GPT-4.1, GPT-4o), Anthropic (Claude 3.5), Google Gemini (2.0 Flash). Switch providers anytime with novax configure.

Live Demo
Multi-tool workflows
The AI chains tools together autonomously. One natural language request triggers multiple Android actions.
nova — surroundings query
You: What can you tell me about my surroundings?
───────────────────────────────────────
⚡ analyze_photo(camera_id=0)
  → /sdcard/nova_20250714_143201.jpg
⚡ get_location()
  → 12.9716°N, 77.5946°E (±8m)
⚡ get_sensor(sensor="light")
  → 847 lux
───────────────────────────────────────
◆ You're in a well-lit room in Bengaluru,
  India. Back camera shows a desk with a
  laptop. Light level 847 lux — bright
  enough for reading or video calls.
nova — break reminder
You: Dim to 20%, lower volume, remind me to take a break
───────────────────────────────────────
⚡ set_brightness(level=51)
  → Brightness → 51 (20%)
⚡ set_volume(stream="music", level=20)
  → Volume → 20%
⚡ send_notification(title="⏰ Break",
   message="5-minute break time!")
  → Notification sent ✓
───────────────────────────────────────
◆ Done. Screen dimmed, music lowered,
  and break reminder pushed.
🧠 Persistent Memory

The agent remembers you

History saved to ~/.nova_agent/history.json. Close the terminal and come back — context persists across sessions.

Day 1 —
You: My name is Alex. I'm in Mumbai.
Nova: Nice to meet you, Alex! 👋
 
Day 2 — new session —
You: What's my location again?
Nova: You're in Mumbai — you told me yesterday!

Get Started
Up and running in 5 minutes
Works on any Android 7.0+ phone. No root, no ADB, no PC needed.
1
Install Termux + Termux:API from F-Droid
Use F-Droid — the Play Store versions are outdated and missing features.
↗ Install Termux ↗ Install Termux:API
2
Run the installer in Termux
One command installs everything: Node.js, Python, and all dependencies.
$ curl -fsSL https://raw.githubusercontent.com/lekhanpro/nova-agent/main/install.sh | bash
Or via npm: pkg install nodejs-lts -y && npm install -g nova-agent
3
Configure your API key
Interactive wizard: choose provider, model, and paste your key.
$ novax configure
4
Start asking
One-shot queries or persistent interactive chat.
$ novax ask "What's my battery level?"
→ Battery: 87% | Discharging | Temp: 32.1°C
$ novax start # interactive chat with memory
Real Examples
Ask anything about your phone
Natural language. Real Android actions. On-device.
📷 Vision
"Take a selfie and describe what I look like"
Front camera → base64 JPEG → AI vision → returns detailed description of your appearance and surroundings
📍 Location
"Where am I and is my WiFi signal good?"
GPS + WiFi info → city/area + Maps link + signal strength assessment for video calls
💬 Messaging
"Read my last 5 SMS and summarize them"
Reads recent messages → AI extracts key info, senders, action items, and sends you a notification summary
🎚️ Device
"Set volume to 20%, dim screen, turn off torch"
Chains 3 tools: set_volume + set_brightness + torch — all executed in one response
📋 Clipboard
"Fix the grammar in my clipboard text"
Reads clipboard → AI fixes grammar and style → pastes corrected version back
📞 Calls
"Who called me today that I missed?"
Reads call log → filters missed calls from today → returns names/numbers with timestamps
